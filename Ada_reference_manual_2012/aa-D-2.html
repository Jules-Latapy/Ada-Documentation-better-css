D.2 Priority Scheduling

1/3
{AI95-00321-01} {AI05-0299-1} [This subclause clause describes the rules that determine which task is selected for execution when more than one task is ready (see 9 9.2). The rules have two parts: the task dispatching model (see D.2.1), and a specific task dispatching policy (see D.2.2).]
